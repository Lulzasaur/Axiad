## Introduction

This is a very simple full stack React app. 

## Set Up

In backend folder, we will create a database named 'axiad' and feed it seed data. Then, we will start the server using nodemon:

```shell
createdb axiad
node seed.js
nodemon server.js
```

In frontend folder:

```shell
npm start
```
